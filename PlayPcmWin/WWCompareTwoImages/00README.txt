■ Download and install AdobeRGB and Rec.709 color profiles

Download Rec.709 color profile from the following URI:
http://www.color.org/chardata/rgb/BT709.xalter

Right click downloaded ITU-RBT709ReferenceDisplay.icc and select "Install Profile" to install it.

Download AdobeRGB color profile from the following URI's "ICC profile download for End Users":
https://www.adobe.com/support/downloads/iccprofiles/iccprofiles_win.html

Extract all of downloaded zip, right click AdobeRGB1998.icc on the RGB subfolder and install it.

■ Download and install color profile of your monitor.

Download and install color profile of your monitor from the vendor website.

Open Control panel > Color Management, confirm your monitor profile is associated with the monitor.
